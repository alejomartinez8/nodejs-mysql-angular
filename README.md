# App in NodeJS, MySQL, Angular - @alejomartinez8

## Installation

### API - Server

```
$ cd /server
$ npm install
$ npm run dev
```
### DB Config

Create DB and tables with sql files
/model/adviser.example.sql
/model/user.example.sql

### Client - Angular

```
$ cd /client-angular
$ npm install
$ npm install -g @angular/cli
$ ng serve
```
